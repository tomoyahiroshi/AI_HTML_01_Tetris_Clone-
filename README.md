# AI HTML Tetris Clone

ブラウザで遊べるシングルプレイのテトリス実装です。`index.html` 単体で動作し、7-Bag、ホールド、ゴースト表示、スコア/レベル管理など基本機能を備えています。

## 特徴
- 10x20 のテトリス盤面（内部バッファ行あり）
- 7種類ミノ（I, O, T, S, Z, J, L）
- SRS準拠の回転と壁蹴り
- ホールド機能（1ミノにつき1回）
- Next表示（5個）
- スコア / レベル / ライン / ハイスコア表示
- `localStorage` を利用したハイスコア保存

## 操作方法
- `←` / `→` : 左右移動
- `↓` : ソフトドロップ
- `Space` : ハードドロップ
- `Z` / `X` / `↑` : 回転
- `C` / `Shift` : ホールド
- `P` / `Esc` : 一時停止
- `Enter` : ゲーム開始/再開

## 実行方法
1. このリポジトリをクローン
2. `index.html` をブラウザで開く

簡易サーバーで起動する場合の例:

```bash
python3 -m http.server 8000
```

その後、`http://localhost:8000` にアクセスしてください。
